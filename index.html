About this site

The entire page is made up of "panel" elements, each of which can be assigned an optional "size" modifier (satisfying the fixed width requirement). For panels that don't use a size modifier, individual containing elements *inside* them (eg. a column) can be assigned a "span" modifier to give those a fixed width instead (also satisfying the fixed width requirement).

What's fun about it?

Another fun quirk of side-scrolling pages is how to actually implement horizontal scrolling *without* resorting to using the (usually ugly) horizontal scrollbar.
